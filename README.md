# Veritas Metal — Website Oficial

Site institucional da Veritas Metal — empresa de construção com Steel Frame em Marechal Cândido Rondon, Oeste do Paraná.

**Posicionamento:** Construção Inteligente. Engenharia Moderna.

## Stack

| Camada | Tecnologia |
|--------|-----------|
| Framework | Next.js 15 (App Router) |
| Linguagem | TypeScript 5 (strict) |
| Estilos | Tailwind CSS 4 (CSS-first) |
| Scroll | Lenis 1 + GSAP ScrollTrigger |
| Animações | GSAP 3 + Framer Motion 11 |
| Deploy | Vercel |

## Desenvolvimento

```bash
npm install
npm run dev
```

## Scripts

| Comando | Ação |
|---------|------|
| `npm run dev` | Servidor de desenvolvimento (Turbopack) |
| `npm run build` | Build de produção |
| `npm run lint` | ESLint |
| `npm run typecheck` | TypeScript sem emitir arquivos |
| `npm run format` | Formatar com Prettier |
